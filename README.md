# PlayJucs Game Hub 🎮

A cosmic-themed game hub website built with HTML, CSS, and JavaScript.

## Features

✨ **Stunning Design**
- Galactic background with animated star field
- Neon cyan and purple color scheme with glowing effects
- Smooth animations and transitions
- Responsive design for all devices

🎮 **Interactive Features**
- Game category filtering (Action, Retro, Strategy)
- Hover effects on game cards
- Mobile-friendly interface
- Smooth transitions and animations

## Project Structure

```
playjucs-game-hub/
├── index.html       # Main HTML file
├── style.css        # Stylesheet
├── script.js        # JavaScript functionality
└── README.md        # This file
```

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. Use the filter buttons to browse games by category
4. Hover over game cards to see interactive effects

## Customization

### Adding More Games

Edit `index.html` and add new game cards:

```html
<a href="#" class="game-card" data-category="action">
    <div class="game-thumb">🎮</div>
    <div class="game-info">
        <div class="game-title">Game Name</div>
        <div class="game-tags">
            <span class="tag">Action</span>
        </div>
    </div>
</a>
```

### Changing Colors

Edit `style.css` and modify the color values:
- Primary color: `#00ffff` (cyan)
- Secondary color: `#bd00ff` (purple)
- Background: `#050515` (dark blue)

### Adding Game Links

Replace `href="#"` with your game URLs or paths.
